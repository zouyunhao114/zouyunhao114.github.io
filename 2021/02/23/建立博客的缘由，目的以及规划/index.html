建立博客的缘由，目的以及规划

发表于 2021-02-23 更新于 2021-02-24 阅读次数：

这是我的第一篇博客，我会简单的陈述我为什么要建立这个博客和未来的规划。如果你想了解我，可以移步屏幕上方，点击这里即可，这样也许会使你更好的理解接下来的内容。

缘由

其实缘由还是很简单的，以前在网上搜索技术资料时，经常会阅读其他博主及大神的博客，我也萌生过建立博客的想法，无奈想到自己学术，技术水平，时间有限，所以一直只是个想法而已。这个学期是我研究生的最后一学期，前几学期以及把毕业所需要的学分全部修完，论文的数据分析部分也得到了教授的肯定，所以时间，精力这个条件已经满足了。并且，在搜索资料时，恰巧打开了曼亚灿的个人网站, 根据他发表的文章得知同样是非科班出身的他，能做出这么精致的网站，顿时心生敬意，因此我也决定尝试一下！（网站左侧sidebar里有他的网站链接，感兴趣的朋友可以去拜访）

目的

正如之前提到的，由于我的学术，技术水平有限，需要学习的东西还有很多。所以，建立此网站的目的有以下三点， 并且始终贯彻一个中心：分享知识，一起学习。

  1. 我将在我的学习过程中整理的技术知识或者有价值的知识记录下来，方便大家和我查阅，学习，交流。
  2. 如有好的文章会转载并标明出处。
  3. 满足自己的兴趣，勉励自己。

规划

本网站的内容将会以以下几点为主。我也是在学习的阶段，欢迎大家积极留言讨论～

  1. python
  2. 数据分析，数据挖掘
  3. 机器学习(Machine Learning)
  4. 统计学
